◆ apWithFlags

unsigned short CardConstInfo::apWithFlags

A bitset containing the default AP and some flags. apWithFlags & 0x1FFF is the default AP. apWithFlags & 0x2000 is if the card has an image in DATA/PICPACK.MRG. apWithFlags & 0x4000 is if the card's password will work. apWIthFlags & 0x8000 is if the card can be received from reincarnation.

◆ dpWithFlags

unsigned short CardConstInfo::dpWithFlags

A bitset containing the default DP and some flags. dpWithFlags & 0x1FFF is the default DP. dpWithFlags & 0x2000 is if the card appears in slots normally. dpWithFlags & 0x4000 is if the card is a slot rare, with the exception of Brain Control. This is ignored for the special slot rares. dpWIthFlags & 0x8000 is not understood.
