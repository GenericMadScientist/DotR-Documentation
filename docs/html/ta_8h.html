bool Ta_SideMatch ( int  sideA,
int  sideB,
SideComp  compMode 
)

Return if two sides match according to compMode.

If compMode is ALWAYS_MATCH, the result is true. If compMode is MATCH_IF_EQUAL, the result is sideA == sideB. If compMode is MATCH_IF_NOT_EQUAL, the result is sideA == !sideB. If compMode is NEVER_MATCH, the result is false. If compMode is anything else, the result is false.

Parameters
sideAThe first side to compare
sideBThe second side to compare
compModeThe way to compare the two sides
Returns
Whether the two sides match
